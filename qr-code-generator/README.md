# QR Code Generator

A Python project to generate QR codes from text or URLs.

## Features
- Convert text and URLs into QR codes
- Save QR codes as images
- Easy-to-use interface

## How to Use
Run the script and provide the text or URL you want to convert to a QR code.

```bash
python qr_code_generator.py
```

## Files
- `qr_code_generator.py` - Main QR code generator script

## Requirements
Make sure to install the required library:
```bash
pip install qrcode[pil]
```
