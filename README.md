# Code with Python 🐍

Welcome to Code with Python! This repository contains beginner-friendly Python projects designed to help you learn programming concepts through practical examples.

## Projects

### 1. [Password Generator](./password-generator/)
Learn how to create a program that generates secure passwords with customizable options.

### 2. [QR Code Generator](./qr-code-generator/)
Discover how to convert text and URLs into QR codes that can be scanned by any device.

## Getting Started

1. **Clone this repository:**
   ```bash
   git clone https://github.com/Dannnnie/Codewithpython.git
   cd Codewithpython
   ```

2. **Navigate to a project folder:**
   ```bash
   cd password-generator
   # or
   cd qr-code-generator
   ```

3. **Follow the project's README** for specific instructions.

## Requirements
